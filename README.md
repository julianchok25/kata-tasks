# Kata - Task Management

## Local Installation

Frontend and Backend requires [Node.js](https://nodejs.org/) v20+ to run.

Install the dependencies and start the server on **each terminal**.

**New Terminal 1**
```sh
cd backend
npm i
npm run dev
```

**New Terminal 2**
```sh
cd frontend
npm i
npm run dev
```
## Tech Stack
* Node.js v20+
* ExpressJs
* MongoDB
* ReactJs
* Typescript
* Jest

## Default Ports
* backend: 3000
* frontend: 5173
